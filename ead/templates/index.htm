{% trans "Welcome to the Egyptian Archaeological Database (EAD)" %}

{% blocktrans %}The EAD is a bilingual online database of Egyptian archaeological sites from pre-dynastic times through Byzantine/Coptic times. Its primary purpose is to enable preservation and conservation of these sites. The database presently includes over 1500 sites ranging from small Palaeolithic workstations to large complexes such as Abydos and Tall al Dabah (Avaris).


In this web site we are demonstrating the EAD with a very small selection of 10 sites in order to show the type of data, bibliography and imagery that is being collected for all sites.


We welcome your comments and ideas to improve the type of information we have collected. Please send them to our email address below. Thank you very much for your help!


The EAD is powered by Arches, an open-source, standards-based, web platform developed by the Getty Conservation Institute and the World Monuments Fund. TMP customized Arches version 3.1 to better meet the requirements of Egyptian archaeology, with the help of students from AUC and Legion GIS LLC and supported by ARCE. We would particularly like to thank Adam Cox of legiongis for allowing us to host this demonstration web site.{% endblocktrans %}

{% trans "How do I use the EAD?" %}

{% blocktrans %}The EAD has 3 pages for users to interact with:

Home Page

You may view the EAD in English or Arabic. If you wish to switch to Arabic, select the small box at the top right of the page marked 'English', choose 'Arabic' and click ‘Change’ in the box next to it.{% endblocktrans %}


{% trans "Map View" %}

{% blocktrans %}The map view displays all the archaeological sites in orange on a Bing Satellite map of Egypt. This map is being regularly updated by Bing. The geographic position of the cursor is displayed at the bottom left of the screen as latitude and longitude.

You can see an overview of the sites in Egypt or explore one area in more detail by zooming in or out using the + and – controls on the top left of the map.

The map view is a composite of a base map and layers which contain the archaeological sites. The type of base map and layer being displayed is controlled by the white boxes at the bottom of the map.

The default base map is a hybrid of the Bing satellite map and the Bing Street map. But it can be changed to show the satellite map or just the streets map, by selecting the base map button. Selecting the name of the map will turn it on or off. For ancient Egypt, the satellite map is the most useful.

The overlays button shows the layers currently displayed and allows these to be turned on or off. Multiple ones may be turned on together. There is a layer for each type of resource: archaeological sites and archaeological complexes.

Hovering over a site marker will show the primary name of the site.

Clicking the marker will display a pop-up window showing some basic details of the site: the site function and its chronology and location in degrees latitude and longitude. The box also contains a button for viewing the site report.{% endblocktrans %}


{% trans "Search" %}

{% blocktrans %}The search screen enables you to search for a particular site or a group of sites sharing a characteristic, such as site type or cultural period.

When selecting the search page, you will see a number of search boxes for direct links to particular sites or groups of sites. For instance, if you select ‘Abu Shaar’ you will be taken directly to the report on Abu Shaar showing all the information we have on that site to date. If you select ‘All Sites’, a list of all sites in the demo database will be displayed and you may select one to find the detailed data on that site.

If, for some reason, you lose the search boxes, simply select Tools> Popular Searches (on the top right of the page), and they will be displayed again.

For this demo system, please ignore the various options to export search results.

Clicking the ‘Location Filter’ green box at the top of the page will show a map of Egypt displaying all the sites in the database. This is a toggle switch - it may be switched off by clicking again. If a particular search is being made while the location filter is on, only those sites that are displayed in the search results will also be displayed on the map.

One of the features of the EAD is the ability to search for sites using variant names. To try this out, you can input a name that you would commonly use for one of the sites, into the long box marked ‘Search for Resource’ at the top of the page. A list of possible site names and other information will appear. If the name is in our database, select the name in the list (with name.E41 in parentheses) and you will find a brief description of that name under our primary name. Select this for the full report on the site. This search box may also be used for setting up searches such as searching for all New Kingdom temples. We have created a search for all sites with a Ptolemaic temple as an example of how to do this {% endblocktrans %}


{% trans "Site Reports" %}

{% blocktrans %}The site reports display the detailed information we have available for a site or complex and provides internal links to the bibliography for the site, images we may have and cross references to other sites.

The satellite image at the top of the report shows the site’s location (blue circle). You may use the zoom controls at the top left (+/-) to either view the surrounding area or focus on the site itself. This image will be the most recent satellite image of the site taken by Bing.

If the site’s location is known, it is given in degrees, minutes and seconds, and degrees decimal. To view the site in Google Maps, which enables you to view a 3-D image and may provide further information on the surroundings, click below the coordinates.

The section titled ‘Related Archaeological Sites’ provides a clickable list of cross references to other sites in the EAD or other sites within a complex, such as an oasis.

You can go to the bibliography for the site by clicking on the name in the ‘Related Documents’ section. You will be taken to the bibliography record first, then select the option 'link to document' to view the bibliography itself. This is a PDF file which may be printed out. Use the back arrow to return to the site.

Images for the site are shown as thumbnails in the ‘Related Images’ section at the end of the report. Clicking a thumbnail displays the image, a description and its provenance.

In order to print out the report, click the button at the top left. A PDF of the report is produced for sending to a printer.{% endblocktrans %}
